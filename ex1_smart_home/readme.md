# Importare il progetto in itemis CREATE

Breve guida in italiano per importare il progetto in itemis CREATE.

## Requisiti
- Java JDK compatibile (ad es. 11).  
- itemis CREATE 

## Importazione
1. Apri itemis CREATE.  
2. File > Import...  
3. Scegli "General > Existing Projects into Workspace" e clicca Next.  
4. Seleziona "Select root directory" e punta alla cartella del progetto.  
5. Seleziona i progetti trovati e clicca Finish.
